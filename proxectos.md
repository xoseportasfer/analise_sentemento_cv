1. Clasificador de Tickets de Soporte al Cliente
Ideal para empresas que reciben miles de correos. El programa extrae el departamento (Ventas, Técnico, Facturación), la prioridad (Alta, Media, Baja) y el sentimiento del cliente para priorizar quejas.

2. Monitor de Reputación de Marca en Redes Sociales
Adaptas el código para procesar comentarios de Twitter o Instagram. Extrae si la mención es una queja, elogio o sugerencia, y clasifica el sentimiento para generar alertas automáticas ante crisis de reputación.

3. Filtro de Spam y Contenido Inapropiado para Foros
Un sistema que analiza mensajes y extrae categorías de toxicidad (insultos, spam, contenido adulto, discurso de odio). Demuestra que sabes construir sistemas de moderación automática.

4. Analizador de Intención de Compra en Ventas
Utiliza el código para analizar transcripciones de llamadas de ventas o chats. El modelo clasifica si el usuario está en fase de "Investigación", "Comparación" o "Listo para comprar", ayudando al equipo de ventas a cerrar tratos.

5. Clasificador Automático de Noticias Geopolíticas
Similar a tu ejercicio de prensa. El programa toma artículos y extrae el país mencionado, el tema principal (conflicto, economía, salud) y la tendencia ideológica del medio que escribe.

6. Sistema de Triaje para Currículums (HR Tech)
En lugar de texto político, procesas resúmenes de CVs. El sistema clasifica el nivel de experiencia (Junior, Senior), la tecnología principal (Python, Java) y la disponibilidad del candidato.

7. Extractor de Requisitos en Documentos Legales
Adaptas el BaseModel para buscar cláusulas específicas en contratos. Clasifica el tipo de contrato, extrae la fecha de vencimiento y detecta si hay cláusulas de penalización presentes.

8. Analizador de Feedback en Educación (EdTech)
Procesa comentarios de alumnos sobre cursos. Extrae la dificultad percibida (Muy fácil, Adecuada, Muy difícil), el recurso más valorado (vídeos, tareas) y el idioma del estudiante.

9. Detector de "Fake News" o Sesgo Mediático
Un programa que analiza un párrafo y clasifica la objetividad del texto, detectando si utiliza lenguaje emocional/sensacionalista o si presenta un sesgo partidista marcado.

10. Organizador de Gastos Personales (FinTech)
Al pasarle una descripción de un gasto (ej: "Cena en el restaurante italiano 50€"), el modelo clasifica la categoría (Alimentación, Ocio, Transporte) y detecta la moneda y el monto total.



Sector Salud y Bienestar
Triaje de Síntomas para Telemedicina: Analiza descripciones de pacientes para extraer la urgencia (Urgente, Consulta, Seguimiento), la especialidad médica necesaria y los síntomas clave.

Analizador de Etiquetas Nutricionales: Procesa listas de ingredientes para detectar alérgenos (gluten, lactosa), clasificar el tipo de dieta (vegana, keto) y extraer el valor calórico mencionado.

Resumidor de Ensayos Clínicos: Extrae de artículos científicos el tamaño de la muestra, la fase del ensayo (I, II, III), el fármaco estudiado y el resultado principal (positivo, neutro, negativo).

Monitor de Bienestar de Empleados: Clasifica respuestas en encuestas de clima laboral para detectar niveles de burnout, satisfacción con el liderazgo y equilibrio vida-trabajo.

Sector Inmobiliario y Construcción (PropTech)
Extractor de Características de Viviendas: Procesa descripciones de portales inmobiliarios para extraer número de habitaciones, metros cuadrados, extras (piscina, garaje) y estado de la reforma.

Analizador de Normativa Urbanística: Extrae de documentos municipales el coeficiente de edificabilidad, la altura máxima permitida y el uso de suelo (residencial, comercial).

Clasificador de Incidencias en Obras: Analiza partes diarios de obra para categorizar retrasos (climatológicos, falta de material, personal) y el impacto en el cronograma.

Sector Logístico y E-commerce
Gestor de Incidencias de Envío: Clasifica reclamaciones de logística para detectar el punto de fallo (almacén, transporte, última milla) y si se solicita reembolso o reenvío.

Analizador de Reseñas de Producto (Marketplace): Extrae de los comentarios la característica más mencionada (durabilidad, diseño, precio) y si hay comparaciones directas con la competencia.

Extractor de Datos de Facturas de Proveedores: Automatiza la lectura de documentos PDF/imágenes para extraer el CIF, la base imponible, el IVA y la fecha de vencimiento del pago.

Sector Financiero y Seguros (FinTech / InsurTech)
Analizador de Declaraciones de Siniestros: Clasifica reportes de accidentes de coche para determinar la culpabilidad aparente, el tipo de daño (chapa, motor, cristales) y la ubicación.

Detector de Señales de Inversión (Trading): Procesa noticias financieras para clasificar el impacto esperado en una acción (Bullish, Bearish, Neutral) y el sector afectado.

Clasificador de Consultas de Seguros de Vida: Analiza perfiles de clientes para recomendar el tipo de póliza (Riesgo, Ahorro, Jubilación) basado en su edad y situación familiar.

Sector Viajes y Hospitalidad (TravelTech)
Organizador de Itinerarios de Viaje: Toma correos de confirmación de vuelos y hoteles para extraer fechas, códigos de reserva, aeropuertos y nombres de hoteles en un JSON estructurado.

Analizador de Preferencias Gastronómicas: Clasifica comentarios en apps de restauración para extraer platos estrella, nivel de ruido del local y calidad del servicio.

Sector Energía y Sostenibilidad
Auditor de Memorias de Sostenibilidad (ESG): Extrae de informes anuales el porcentaje de reducción de huella de carbono, el consumo de energía renovable y las metas de diversidad.

Clasificador de Anomalías en Redes Eléctricas: Analiza reportes técnicos para categorizar tipos de avería (cortocircuito, sobrecarga, mantenimiento preventivo) y el número de usuarios afectados.

Proyectos Transversales y de Productividad
Asistente de Actas de Reunión: Toma transcripciones y extrae automáticamente los "Action Items", los responsables asignados y las fechas límite discutidas.

Organizador de Bibliotecas Digitales: Analiza párrafos de libros o artículos para extraer el género literario, la época histórica mencionada y el tono narrativo.

Analizador de Consultas de API (Developer Experience): Clasifica tickets de desarrolladores para detectar si el problema es un error de documentación, un bug del SDK o un límite de cuota.
